# Kokkadan Footwear

Kokkadan Footwear is a one-stop shop for:
- **Footwear** - Shoes, Chappals, Slippers, Boots, Safety Shoes
- **Bags** - School bags, Travel bags, Trolleys, Luggage
- **Umbrellas** - Popy, Johns
