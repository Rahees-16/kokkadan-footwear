# Kokkadan Footwear Website

## Project Overview
Static website for Kokkadan Footwear, a family-owned retail shop in Edakkara, Kerala, India. Established in 1975, serving the community for 50+ years.

## Tech Stack
- HTML5, CSS3, JavaScript (vanilla)
- No frameworks - pure static site
- Hosted on GitHub Pages

## Directory Structure
```
kokkadan-footwear/
├── css/
│   └── styles.css       # Main stylesheet with CSS variables
├── js/
│   └── main.js          # Mobile menu, filters, smooth scroll
├── images/
│   ├── brands/          # Brand logos (if local)
│   ├── hero/            # Hero section images
│   └── products/        # Product images (if local)
├── index.html           # Home page
├── footwear.html        # Footwear catalog
├── bags.html            # Bags & trolleys catalog
├── sports.html          # Sports goods catalog
├── umbrellas.html       # Umbrellas catalog
├── about.html           # About us page
├── contact.html         # Contact page with map
└── README.md            # Project readme
```

## Key Features
- Responsive design (mobile-first)
- WhatsApp integration for inquiries
- Product filtering by category
- Google Maps integration
- Brand showcase sections

## Color Scheme
- Primary: #6B2D2D (Maroon)
- Secondary: #D4A847 (Gold)
- Background: #F9F6F0 (Cream)

## Products Sold
- **Footwear**: Bata, VKC, Relaxo, Sparx, Paragon, Walkaroo, Lunars, De Bongo, Leeds, Aqualite, and more
- **Bags**: Skybags, American Tourister
- **Umbrellas**: Popy, Johns, Lala
- **Sports**: Cricket, Football, Badminton equipment
